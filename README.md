# Ansible How-to

Set up virtual environment:

```sh
uv venv
```

Activate virtual environment:

```sh
source .venv/bin/activate
```

Install dependencies:

```sh
uv sync
```

View notebook in read-only mode:

```sh
marimo run how_to.py
```

Or edit the notebook, create new notebooks, etc.:

```sh
marimo edit
```
